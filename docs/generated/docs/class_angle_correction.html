Detailed Description

Add a value to an angle input (in radians). The output value is wrapped to a range between [0, 2*pi]. This transform is useful for correcting e.g. a heading or a wind direction reading.

Parameters
offsetValue to be added, in radians.
min_angleIf you have output between -pi and pi, use -3.14159265, otherwise use 0. Default value is 0.
config_pathPath to configure this transform in the Config UI.
